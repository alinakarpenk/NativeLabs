### Лаборатораторна №1

#### Інструкція запуску

```
git clone https://github.com/alinakarpenk/ReactNativeLabs.git
npm install
npm start
npx expo start
```

#### У додатку реалізовано:
- Екран клікера
- Екран викликів
- Табова навігація

#### Сторінка із завданнями:
* Зробити 10 кліків – Натиснути на об'єкт 10 разів.
* Зробити подвійний клік 5 разів – Використати TapGestureHandler
для виконання 5 подвійних кліків.
* Утримувати об'єкт 3 секунди – Використати
LongPressGestureHandler для довгого натискання.
* Перетягнути об'єкт – Використати PanGestureHandler, щоб
перемістити об'єкт по екрану.
* Зробити свайп вправо – Використати FlingGestureHandler, щоб
зробити швидкий свайп вправо.
* Зробити свайп вліво – Використати FlingGestureHandler, щоб
зробити швидкий свайп вліво.
* Змінити розмір об'єкта – Використати PinchGestureHandler, щоб
збільшити або зменшити об'єкт.

#### Основні функції `useClikerFunctions`

* `tap` - Обробляє одинарний дотик до елемента.
* `doubleTap` - Обробляє подвійний дотик.
* `longPress` - Спрацьовує при довгому натисканні на елемент.
* `pan` - Відповідає за перетягування елемента по екрану.
* `flingRight / flingLeft` - Обробляє свайп вправо та вліво.
* `pinchGesture` - Обробляє жест масштабування двома пальцями.
