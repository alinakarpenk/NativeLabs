### Лаборатораторна №1

#### Інструкція запуску

```
git clone https://github.com/alinakarpenk/ReactNativeLabs.git
npm install
npm start
npx expo start
```


#### У додатку реалізовано: 

- перегляд списку постів;
- перегляд галереї зображень;
- форма для введення користувацьких даних;
- навігація між екранами за допомогою вкладок;
- використання компонентів React Native та стилізації.

#### Основні використані компоненти:

`FlatList` - компонент який використовується для відображення списків, основною перевагою є те, що він відображає тільки ті елементи, які знаходяться на екрані, а інші підвантажуються під час прокрутки.

`NavigationContainer` - це обгортка, яка відповідає за дерево навігації та історію
переходів

`createBottomTabNavigator()` - фабрична функція яка створює навігацію з вкладками (панель з кнопками)

#### Компоненти створені функцією:

`Tab.Navigator` - контейнер для вкладок, який створює нижнє меню
`Tab.Screen` - самі вкладки, додає екрани у це меню
